# Dynamic Pricing Optimization Engine 🚀

This is a Machine Learning capstone project focused on maximizing e-commerce revenue through demand forecasting and price elasticity modeling. 

## 📂 Current Project Structure
* **`src/`**: Contains the core machine learning logic.
  * `data_generator.py`: Generates synthetic market data.
  * `train_model.py`: Trains the predictive Linear Regression model.
  * `optimize_price.py`: The prescriptive algorithm to find the maximum revenue price point.
* **`api/`**: (In Progress) The FastAPI backend to serve the model.
* **`data/`**: Ignored via `.gitignore` to protect raw data.

## 🛠️ Tech Stack
* Python
* Pandas & NumPy
* Scikit-Learn
* FastAPI
* Streamlit
